How does it work? (Don't forget to reload between testing the functions)

Add stream
Select an event by clicking on the green button with the "+" icon in the event's row
You can refine the stream list via the category selector on the top right corner (won't work here but you can imagine)
Select a stream to enable the "Add stream" button
The edited row gets a minor flash, the "add" button disappears, "edit" and "delete" buttons appear (not implemented either)
Edit stream
Click the button with the wrench icon to toggle the edit panel
Choose another stream to enable the "Update stream" button
Delete stream
Click the red button to delete the stream which is attached to an event
After clicking the "Go on!" button in the dialog the buttons in the row of the event will be updated he other way around like they do after "Add"
Further enhancements
A datepicker to refine events by timestamp for the event list
Sort columns by "A-Z" or by timestamp
Clicking on a category label in the list of streams refines the list to that certain category
